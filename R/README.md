# Introduction to R

To run R from the command line run

```
Rscript R/hello_world.R
```

and/or

```
Rscript -e "print('Hello World')"
```

to output run as `Rscript -e "print('Hello World')" > out.txt`. 

Note: Path to R and Rscript may be: `C:/'Program Files'/R/R-4.2.2/bin/`.
